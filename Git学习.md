## 学习内容
### A下载及布置Git
下载的时候官网上一直转圈圈，点不动下载按钮。于是找了mirror把Git下下来了，但是下了之后才发现还没有vscode，于是又爬回去下，下载速度实在是赛蜗牛。
下载好这些东西并设置好后就可以在vscode里面编辑啦。

### B
vscode使用起来还是挺好的，虽然中间出现了上传不了的情况，但后面更换了一个上传仓库就好了（原因有很多种，比如合并冲突、拼写错误、重复上传等等，尚在思考中...)
Git Bash操作比较复杂，本人不是很习惯。
在仓库中直接上传感觉最方便，目前没感受到弊端。
