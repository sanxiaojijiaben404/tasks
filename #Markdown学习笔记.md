## 学习内容

### 基本定义

Markdown 是一种轻量级的标记语言,可以用纯文本格式编写文档，同时能轻松转换为结构化的 HTML 或其他格式（如 PDF、Word）

### 基本语法

#### 标题语法

标题分为一级标题到五级标题,类型的值是#标题名称,**等级不同,#的数目就不同**

> 一级标题：# 你好
>
> 五级标题：##### 你好

#### 段落语法

段落需要用空白行将一行或多行文本进行分隔

> 不要用空格或制表符缩进段落。

| i like you |
| ---------- |
|            |
| i like you |



#### 换行语法

在一行的末尾添加两个或多个空格或者一个换行或者一个空行

> 一个换行就是<br>

#### 强调语法

强调分为**粗体**和*斜体*

若要加粗，需加粗部分的两侧各添加两个星号,<br>

若要斜体，需在斜体部分的两侧各添加一个星号,<br>

**既要加粗又要斜体，需在该部分的两侧各添加三个星号**.

#### 引用语法

要创建引用，请在段落前添加一个 > 符号

> 第一个

| > life |
| ------ |

> 第二个

| >life     |
| --------- |
| >         |
| > passion |

> 第三种

| > #### The quarterly results look great!       |
| ---------------------------------------------- |
| >                                              |
| > - Revenue was off the chart.                 |
| > - Profits were higher than ever.             |
| >                                              |
| >  *Everything* is going according to **plan** |

#### 列表语法

要创建有序列表，请在每个列表项前添加数字并紧跟一个**英文句点**或者破折号 (-)、星号 (*) 或加号 (+)

> 数字不用按照从小到大的顺序

| 1. First item<br/>2. Second item<br/>8. Third item<br/>    1. Indented item<br/>    2. Indented item<br/>7. Fourth item |
| ------------------------------------------------------------ |

#### 代码语法

要将单词或短语表示为代码，请将其包裹在反引号 (`) 中。

> 类似'code'

> **若要创建代码块，请将代码块的每一行缩进至少四个空格或一个制表符**

#### 分隔线语法

创建分隔线使用1个或多个星号***、---或下划线 ___.

> **注意:分隔线的前后均添加空白行**

#### 链接语法

| 链接表示                   |
| -------------------------- |
| 1.[链接名](链接地址 )      |
| 2.句子[链接名字](链接地址) |
| 3.<URL>或者<email>         |

> 句子可以随意编辑,也可以把句子改成强调字符等

#### 图片语法

| --------------------- |
